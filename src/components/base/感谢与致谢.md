## 🔧 二次开发说明

本项目在 gi-component 的基础上进行了以下优化和改进：

1. **国际化支持** - 适配项目的国际化需求
2. **主题定制** - 根据项目设计规范调整主题色彩
3. **功能增强** - 添加业务特定的功能特性
4. **布局优化** - 优化搜索表单布局（按钮右下角对齐）
5. **性能优化** - 提升组件渲染性能
